## JavaScript функциональность

### 1. Плавная прокрутка по якорям
- `scrollToBooking()` - быстрая прокрутка к форме записи
- Все якоря в меню используют плавную прокрутку
- Поддержка закрытия мобильного меню при клике на ссылку

### 2. Валидация формы
- `validateName()` - проверка имени (минимум 3 символа)
- `validatePhone()` - проверка номера телефона (10+ цифр)
- `validateEmail()` - проверка email адреса
- `validateBookingForm()` - основная валидация всей формы

### 3. Отправка формы
- Отправляет данные после валидации
- Пока показывает alert с подтверждением
- Готова к интеграции с backend API (закомментированные примеры в коде)
- Автоматически очищает форму после отправки

### 4. Форматирование телефона
Автоматическое форматирование при вводе:
- +7 (999) 999-99-99

### 5. Анимация при прокрутке
- Использует Intersection Observer API
- Добавляет класс 'visible' к элементам при прокрутке
- Плавная визуализация появления элементов
