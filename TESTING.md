# Инструкция по тестированию сайта Anasteisha

## Запуск проекта

```bash
cd C:\Users\Ксения\Yandex.Disk\PythonProjects\web_projects\beauty
python -m uvicorn backend.app.main:app --host 0.0.0.0 --port 8000 --reload
```

Сайт доступен по адресу: http://localhost:8000

---

## 1. Приветственное окно для новых посетителей

**Как тестировать:**
1. Откройте браузер в режиме инкогнито (Ctrl+Shift+N в Chrome)
2. Перейдите на http://localhost:8000
3. Через 2 секунды должно появиться модальное окно со скидкой -15%

**Что проверить:**
- [ ] Окно появляется только для новых посетителей
- [ ] Кнопка "Записаться со скидкой" скроллит к форме записи
- [ ] Кнопка "Посмотреть услуги" закрывает окно
- [ ] Кнопка закрытия (×) работает
- [ ] При повторном посещении (без инкогнито) окно не появляется

**Сброс для повторного тестирования:**
В консоли браузера (F12 → Console):
```javascript
localStorage.removeItem('welcomeShown');
localStorage.removeItem('welcomeShownDate');
location.reload();
```

## Полезные команды

**Очистить localStorage полностью:**
```javascript
localStorage.clear();
location.reload();
```

**Проверить текущую тему:**
```javascript
console.log('Тема:', localStorage.getItem('theme') || 'системная');
```

**Проверить статус приветственного окна:**
```javascript
console.log('Показано:', localStorage.getItem('welcomeShown'));
console.log('Когда:', localStorage.getItem('welcomeShownDate'));
```
