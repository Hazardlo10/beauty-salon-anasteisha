# Режим технических работ

Этот файл описывает как включить/выключить режим "Ведутся технические работы" на сайте Anasteisha.

---

## Быстрый старт

### Включить режим тех. работ:

Откройте файл `frontend/static/js/script.js` и измените строку 9:

```javascript
const MAINTENANCE_MODE = true;  // Включить
```

### Выключить режим тех. работ:

```javascript
const MAINTENANCE_MODE = false;  // Выключить (по умолчанию)
```

---

## Настройка времени окончания работ

Вы можете указать ожидаемое время окончания работ. Откройте `frontend/static/js/script.js` и измените строку 12:

```javascript
// Примеры:
const MAINTENANCE_END_TIME = '2026-02-03 10:00';  // 3 февраля 2026, 10:00
const MAINTENANCE_END_TIME = '2026-02-05 18:30';  // 5 февраля 2026, 18:30
const MAINTENANCE_END_TIME = null;                 // Не указывать время (по умолчанию)
```

Формат: `'YYYY-MM-DD HH:MM'` (год-месяц-день часы:минуты)

## Что видит пользователь

Когда режим технических работ включен, пользователь видит:

1. **Тёмный фон** с анимированными декоративными элементами
2. **Иконку инструментов** с пульсирующим эффектом
3. **Заголовок** "Ведутся технические работы"
4. **Сообщение** о временной недоступности
5. **Время завершения** (если указано)
6. **Контактные данные** для связи:
   - Телефон (кнопка)
   - Telegram (кнопка)
7. **Логотип** Anasteisha

## Изменение контактных данных

Чтобы изменить номер телефона или ссылку на Telegram, откройте `index.html` и найдите секцию `MAINTENANCE MODE` (строки 22-53):

```html
<a href="tel:+79001234567" class="maintenance-phone">
    <i class="fas fa-phone"></i> +7 (900) 123-45-67
</a>
<a href="https://t.me/your_username" class="maintenance-telegram">
    <i class="fab fa-telegram"></i> Telegram
</a>
```

Замените:
- `+79001234567` и `+7 (900) 123-45-67` на реальный номер телефона
- `your_username` на имя пользователя в Telegram

## Альтернативный способ включения

Если нужно включить режим без редактирования JavaScript, можно изменить HTML напрямую.

Откройте `index.html` и найдите строку 24:

**Выключено (по умолчанию):**
```html
<div class="maintenance-mode" id="maintenanceMode" style="display: none;">
```

**Включено:**
```html
<div class="maintenance-mode" id="maintenanceMode" style="display: flex;">
```

## Примеры использования

### Плановое обновление на 2 часа:
```javascript
const MAINTENANCE_MODE = true;
const MAINTENANCE_END_TIME = '2026-02-03 14:00';
```

### Срочные работы без указания времени:
```javascript
const MAINTENANCE_MODE = true;
const MAINTENANCE_END_TIME = null;
```

### Работы завершены:
```javascript
const MAINTENANCE_MODE = false;
const MAINTENANCE_END_TIME = null;
```
