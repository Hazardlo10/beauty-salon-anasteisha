JAVASCRIPT ФУНКЦИОНАЛЬНОСТЬ
================================================================================

1. ПЛАВНАЯ ПРОКРУТКА
   - scrollToBooking() - быстрый переход к форме записи
   - Все якоря в меню имеют плавную анимацию
   - Поддержка закрытия мобильного меню при навигации

2. ВАЛИДАЦИЯ ФОРМЫ
   - validateName() - проверка имени (минимум 3 символа)
   - validatePhone() - проверка телефона (10+ цифр)
   - validateEmail() - проверка email адреса
   - validateBookingForm() - общая валидация всей формы
   - Отображение ошибок в alert диалоге

3. ОТПРАВКА ФОРМЫ
   - submitBookingForm() - обработка отправки
   - Проверка всех данных перед отправкой
   - Показ подтверждения через alert
   - Логирование в браузерную консоль
   - Очистка формы после отправки
   - Готовый код для интеграции с backend API

4. ФОРМАТИРОВАНИЕ
   - Автоматическое форматирование номера телефона
   - Формат: +7 (999) 999-99-99

5. АНИМАЦИЯ
   - Использование Intersection Observer API
   - Плавное появление карточек услуг при прокрутке
